# NAFIS

## NAFIS?

NAFIS stands for
 - *N*ot
 - *A*
 - ***F*orgery**
 - *I*
 - *S*wear

It's based around forging and customizing your weapons, taking some inspiration from, but not plagiarizing Tinkers.
